TargetGroup

Example

resource TargetGroup example { name = "example-value" port = 42 protocol = "HTTP" protocolVersion = "HTTP1" vpcId = "example-value" targetType = "instance" ipAddressType = "ipv4" healthCheck = "..." deregistrationDelay = 300 slowStart = 0 stickiness = "..." targets = ["item1", "item2"] tags = { key = "value" } }

Properties

Name ↕ Type ↕ Default Valid Values Optional ↕ Description
name# string — — Yes The name of the target group
port# integer — — Yes The port on which targets receive traffic
protocol# string HTTP
HTTP HTTPS TCP TLS UDP TCP_UDP GENEVE
Yes The protocol for routing traffic
protocolVersion# string HTTP1
HTTP1 HTTP2 GRPC
Yes The protocol version (ALB only)
vpcId# string — — Yes The VPC ID for the target group
targetType# string instance
instance ip lambda alb
Yes The target type
ipAddressType# string ipv4
ipv4 ipv6
Yes The IP address type
healthCheck# healthcheck — — Yes Health check configuration
deregistrationDelay# integer 300 — Yes Deregistration delay in seconds
slowStart# integer 0 — Yes Slow start duration in seconds (ALB only)
stickiness# stickiness — — Yes Stickiness configuration
targets# list — — Yes Target IDs to register (instance IDs or IP addresses)
tags# map — — Yes Tags to apply to the target group
